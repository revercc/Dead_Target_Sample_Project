# Forward Assault Project
A mod project for Forward Assault. Works on both Android Studio and AIDE

Offsets are outdated. Update it yourself!

# Features:
Player:
- Telekill
- Masskill
- Fly hack (buggy)

Visuals:
- Default Chams
- Wireframe Chams
- Glow Chams
- Outline Chams
- Rainbow Chams
- Line Width 0-255
- Color Red 0-255
- Color Green 0-255
- Color Blue 0-255
- Esp Line
- Esp Box
- Esp Box Type 0-1
- Esp Box Width 0-100
- Esp Color 0-7
- Esp Hight 0-100
- Esp Box Size 0-100

Others:
- Speed hack
- Speed 0-100   
- Enemy Scale X 0-25
- Enemy Scale Y 0-25
- Enemy Scale Z 0-25
- No AntiHack
